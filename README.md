## 🚀 Instrucciones para iniciar:

- Instalación:

  ```bash
  yarn install
  # or
  npm install
  ```

- Ejecutar todos los retos:

  ```bash
  yarn test
  # or
  npm test
  ```

- Ejecutar un reto específico.

  ```bash
  yarn test 2022/reto-XX # donde XX es el número del reto.
  # or
  npm test 2022/reto-XX # donde XX es el número del reto.
  ```
